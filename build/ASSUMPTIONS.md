# Build Assumptions Register (SF-6, per V1-GOV-007 / V3-V3A-004)

**Purpose:** every design-freeze decision (DD-01..30) the build implicitly answers before formal resolution is recorded here as a **time-bound assumption with a review trigger**. Nothing in this file is a decision; each row expires or converts when its trigger fires. The builder may not silently default a DD that is not listed here — add the row first.

**Owner reviews this file at every milestone gate (G0–G5).**
